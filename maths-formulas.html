Conditional Probability
P(A|B) = P(A ∩ B) / P(B)
Independent Events
P(A ∩ B) = P(A) × P(B)
Bayes' Theorem
P(A|B) = P(B|A)P(A) / P(B)
Mean

Median (odd n)

Median (even n)

Most frequent data value
Variance
σ² = Σ(xᵢ − μ)² / n
Standard Deviation
σ = √Variance = √(Σ(xᵢ − μ)² / n)
Basic Limits
limx→0 (sinx)/x = 1
limx→0 (1 - cosx)/x² = ½
limx→∞ (1 + 1/x)x = e
Algebra of Limits
lim (f ± g) = lim f ± lim g
lim (f × g) = lim f × lim g
lim (f / g) = lim f ÷ lim g (g ≠ 0)
Derivatives – Basics
d/dx (xⁿ) = n·xⁿ⁻¹
d/dx (sinx) = cosx
d/dx (cosx) = -sinx
Derivatives – Advanced
d/dx (tanx) = sec²x
d/dx (eˣ) = eˣ
d/dx (lnx) = 1/x
Rules of Differentiation
Chain Rule: d/dx f(g(x)) = f'(g(x))·g'(x)
Product Rule: d(uv)/dx = u'v + uv'
Quotient Rule: d(u/v)/dx = (v·u' - u·v') / v²
Integration Basics
∫xⁿ dx = xⁿ⁺¹ / (n+1) + C (n ≠ -1)
∫1/x dx = ln|x| + C
∫eˣ dx = eˣ + C
Trig Integrals
∫sinx dx = -cosx + C
∫cosx dx = sinx + C
∫sec²x dx = tanx + C
Integration Techniques
∫udv = uv - ∫vdu (by parts)
∫f(ax + b) dx = (1/a)·F(ax + b)
Substitution: Let x = g(t), then use dx = g'(t) dt
Vector Basics
Magnitude: |a| = √(a₁² + a₂² + a₃²)
a·b = |a||b|cosθ
a×b = |a||b|sinθ · n̂
Unit Vector & Direction Cosines
Unit Vector: â = a / |a|
Direction cosines: cosα = x/r, cosβ = y/r, cosγ = z/r
Vector Identities
a × (b × c) = (a·c)b - (a·b)c
(a × b) · c = a · (b × c)
3D Distance & Section
Distance: √[(x₂ - x₁)² + (y₂ - y₁)² + (z₂ - z₁)²]
Section Ratio: (mx₂ + nx₁)/(m+n), etc.
Straight Line (3D)
Vector Form: r = a + λb
Cartesian: (x - x₁)/a = (y - y₁)/b = (z - z₁)/c
Plane Equation
Vector: r · n̂ = d
Cartesian: ax + by + cz = d
Matrix Types
Square, Diagonal, Scalar, Identity
Transpose: Aᵀ, Symmetric: A = Aᵀ
Matrix Operations
(AB)ᵀ = BᵀAᵀ
A⁻¹ = adj(A)/|A|
adj(A)·A = |A|·I
Determinant Properties
|AB| = |A||B|
|Aᵀ| = |A|
Row ops don't change value of |A|

x = |A₁| / |A|
